高速な開発体験

Angular の Signals とスタンドアロンコンポーネントで、シンプルかつ高速に開発できます。

リッチな UI コンポーネント

PrimeNG の豊富なコンポーネントと PrimeUIX テーマで、統一感のあるデザインを実現します。

軽量なネイティブアプリ

Tauri が Rust 製の軽量ランタイムで Web UI を包み、ネイティブに近い動作を実現します。

Windows 向けクロスコンパイル

Linux 環境から cargo-xwin を使って、Windows 向けバイナリをビルドできます。
